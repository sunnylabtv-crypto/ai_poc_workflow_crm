# 🚀 AI Workflow Production

## 빠른 시작

```bash
pip install -r requirements.txt
python config.py development
python main.py --mode health
python main.py --mode monitor
```

## 명령어

```bash
make health         # 연결 테스트
make run            # 단일 실행
make monitor        # 모니터링
make logs           # 로그 확인
```

자세한 설치는 `INSTALLATION.md` 참조
